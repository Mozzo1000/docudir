# docudir
A document management system aimed at being integrated into other systems.

## Proposed API

### Personal area
`/v1/docs/<name>`
`/v1/docs/<name>/<document_id>`

### Collections
`/v1/docs/<name>/collections/<collection_id>`
`/v1/docs/<name>/collections/<collection_id>/<document_id>`

### Anonymized area
`/v1/docs/<document_id>`
